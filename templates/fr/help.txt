Aide pour la liste de diffusion %(listname)s: 

Ceci est l'aide des commandes par courrier pour la version %(version)s
du gestionnaire de listes "Mailman".  Vous trouverez ci-dessous la
description des commandes qui vous permettent d'obtenir des
informations sur vos abonnements et d'en modifier les options pour
les listes de ce serveur.  Chaque commande pour être placée au choix
dans le sujet ou dans le corps du message.

La plupart de ces commandes peuvent être également accomplies par
l'interface web qui se trouve à l'adresse:

    %(listinfo_url)s 

Vous pouvez notamment utiliser cette interface pour vous faire
envoyer votre mot de passe à votre adresse d'abonné. 

Les commandes relatives à une liste (s'abonner, qui y est, etc..)
doivent être envoyées à l'adresse *-request de la liste en question,
par exemple pour la liste 'mailman', utilisez 'mailman-request@...'. 

A propos des descriptions -  les mots entre "<>" sont les éléments 
OBLIGATOIRES tandis que ceux entre "[]" sont des éléments
FACULTATIFS. N'incluez pas les "<>" ou "[]" dans les commandes.

Les commandes suivantes sont disponibles: 

    subscribe [mot-de-passe] [envoi-groupé] [adresse=<adresse>]
        S'abonner à la liste.  Votre mot de passe est nécessaire
        pour vous désabonner ou changer vos options.  Une fois
	abonné à la liste, votre mot de passe vous sera rappelé
        périodiquement.  'envoi-groupé' doit être l'une des deux
        valeurs: 'nodigest' ou 'digest' (sans guillemets!).  Si
	vous souhaitez vous abonner avec une adresse autre que
	celle d'où vous envoyez cette commande, vous pouvez
	indiquer "address=<adresse électronique>" (sans parenthèses
        autour de l'adresse, et sans guillemets!)

    unsubscribe <mot-de-passe> [adresse]
        Se désabonner de la liste. Votre mot de passe doit être
        celui que vous avez donné lors de l'abonnement.  Si vous
	essayez de vous désabonner à partir d'une adresse différente 
	de celle sous laquelle vous êtes abonné, indiquez-la dans 
	le champ 'adresse'.

    who
        Voir tous les abonnés à la liste de diffusion. 

    info
        Obtenir la brève description de cette liste. 

    lists
        Voir quelles sont les listes gérées sur ce serveur Mailman.
 
    help
        Ce message.

    set <option> <on|off> <mot-de-passe> 
        Activer/désactiver des options d'abonnement. Les options
        valides sont: 

        ack : 
	    Recevoir un accusé de réception quand vous envoyez un  
	    message à la liste. 

        digest : 
	    Recevoir un message quotidien regroupant les messages de
            la liste, au lieu de recevoir les messages un par un.

        plain : 
	    Recevoir les envois groupés en texte simple, non MIME
            (uniquement si le mode digest est activé).

	nomail : 
	    Cesser de recevoir les messages.  Utile si vous projetez
            de prendre de courtes vacances.

        norcv : 
	    Activer ceci pour ne PAS recevoir les messages que vous
	    envoyez à la liste.  Ne fonctionne pas si le mode digest
            est activé.

        hide : 
	    Cacher votre adresse lors de la visualisation des abonnés
            de la liste.

    options 
	Montre les valeurs actuelles de vos options pour les listes.

    password <ancien-mot-de-passe> <nouveau-mot-de-passe>
        Changer votre mot de passe.

    end or --
        Cesse le traitement des commandes (utile si votre logiciel de
        messagerie ajoute automatiquement un fichier de signature -
        cela vous évitera bien des ennuis).


Les commandes doivent être envoyées à %(requestaddr)s 

Toute question ou problème à l'attention d'une personne physique doit
être envoyée à 

    %(adminaddr)s
